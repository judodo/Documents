PyPI（全稱為 Python Package Index）是 Python 程式語言的官方第三方軟體套件倉庫。 
截至 2026 年初，PyPI 託管了超過 50 萬個 開源專案，是全球最大的開源軟體庫之一。 

PyPI 的核心功能與角色
集中儲存與分發：開發者將寫好的 Python 程式碼打包成「套件」（Package）並上傳到 PyPI，讓全世界的程式設計師都能搜尋、下載並重複使用這些功能（例如資料分析工具 Pandas 或網頁框架 Django）。
搭配 pip 使用：它是 Python 預設套件管理工具 pip 的後端資料庫。當你在終端機輸入 pip install <套件名稱> 時，pip 就會自動連線到 PyPI 下載並安裝對應的內容。
社群共享：PyPI 由非營利組織「Python 軟體基金會」（PSF）維護，任何人都可以免費註冊帳號並發布自己的開源作品。 

2026 年的最新狀態
規模龐大：目前擁有超過 72 萬個專案、近 800 萬個發布版本，服務全球每年數千億次的下載請求。
安全強化：為了應對日益嚴峻的供應鏈攻擊，2026 年的 PyPI 持續強化安全性，包括全面要求雙重驗證（2FA）以及對第三方擴充模組的安全 API 支援。
效能優化：支援最新的 Python 3.14+ 特性（如自由執行緒 Free-threading），使託管在上面的套件能發揮更佳的並行運算效能。
